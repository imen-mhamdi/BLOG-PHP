
Ce projet est un petit système de blog en PHP.
Il contient l’authentification, la gestion d’articles, un rôle admin et une interface simple.

Fonctionnalités
Connexion / inscription
Profil utilisateur
Ajouter – Modifier – Supprimer un article
Rôles : **Admin** et **User**
Interface responsive (Bootstrap)

Installation

Télécharger le projet


git clone [votre-lien-repo]
cd PHP-BLOG-SYSTEM-main


Créer la base de données

Créer une base : **blog**
Importer : blog.sql


Compte admin

Email : [admin@gmail.com](mailto:admin@gmail.com)
Mot de passe : admin123

Structure rapide


index.php          -> Accueil
login.php          -> Connexion
register.php       -> Inscription
profile.php        -> Profil
blog/              -> Gestion articles
manager.php        -> Config DB
blog.sql           -> Base de données
